flex-basis

Esta propriedade define a largura inicial que um deverá ter. No exemplo abaixo, 100px para a Div1 e 200px para a Div2, desta forma, a largura do item pai (500px) não é utilizada por completo.

                <style>
                    .flex-basis {
                        width: 500px;
                        display: flex;
                        border: 1px solid red;
                    }
            
                    .child-basis1 {
                        border: 1px solid black;
                        flex-basis: 100px;
                    }
            
                    .child-basis2 {
                        border: 1px solid black;
                        flex-basis: 200px;
                    }

Flex-grow

Esta propriedade fraciona a sobra do valor do pai e distribui de acordo com as partes indicadas para cada filho para que seja ocupada a largura total do item pai, nesse exemplo, pai = 500px, filho 1 = 100px e filho 2 = 100px, logo a sobra é igual a 300px, que será divida por 3 e atribuída 2 partes ao filho 1 (200px) e 1 parte ao filho 2 (100px).

                <style>
                    .flex-grow {
                        width: 500px;
                        display: flex;
                        border: 1px solid red;
                    }
            
                    .child-grow1 {
                        border: 1px solid black;
                        flex-basis: 100px;
                        flex-grow: 2;
                    }
            
                    .child-grow2 {
                        border: 1px solid black;
                        flex-basis: 100px;
                        flex-grow: 1;
                    }

Media Query

Um pequeno teste do funcionamento do comando media-query. No comando abaixo foi definido que a cor de fundo da div irá mudar de verde para vermelha e a cor do parágrafo irá mudar para branca, caso a tela fique no tamanho entre 500px e 600px.

                <style>
                    .my-div {
                        background-color: green;
                        height: 20px;
                    }
                    @media (max-width: 600px) and (min-width: 500px) {
                        .my-div {
                            background-color: red;
                        }
                        
                        .my-p {
                            color: white;
                        }
                    }

Faça o teste reduzindo o tamanho da tela para os valores indicados.
